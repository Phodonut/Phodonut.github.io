Snake Revamped

Project Details

-  Mission: Re-create the classic Snake game with a twist

-  Summary: Snake as classicly created but instead of growing whenever you eat food, you grow constantly until you reach the target which restricts your growth. Creates a fast pased highly punishing version, with new dangers and pitfalls.

-  Length: 2 weeks

-  Team Size: 1

-  Platform/Language: Javascript base

-  Notes: Levels not fully implemented in timeline, so they are there as a demonstration of concept, easily implemented in a day if given. WASD movement setup
